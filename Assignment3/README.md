# Voice Translator App

This is a Streamlit-based Voice-to-Voice Translator application.

## Features
- **Speech-to-Text**: Converts your voice to text.
- **Translation**: Translates the text into a selected language.
- **Text-to-Speech**: Plays the translated text as audio.

## Supported Languages
- English
- Hindi
- Tamil
- French
- Arabic

## Setup & Run

1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the App**
   ```bash
   streamlit run app.py
   ```

## Usage
1. Select your **Input Language** and **Output Language** from the sidebar.
2. Click the **Microphone icon** to start recording. Click it again to stop.
3. Wait for the app to transcribe, translate, and generate the audio.
4. Listen to the result!
